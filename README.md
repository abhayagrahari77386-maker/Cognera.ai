# 🚀 Cognera AI – Smart Career Recommendation Platform

> Smarter Choices for a Smarter Career

Cognera AI is an AI-powered career guidance platform that helps students and individuals discover the most suitable career paths based on their interests, skills, personality, and goals. It provides personalized recommendations, skill gap analysis, and structured roadmaps to help users make informed career decisions.

---

## 📌 Problem Statement

Many students struggle to choose the right career due to:

* Lack of proper guidance
* Confusion between multiple career options
* Mismatch between skills and career paths
* No personalized roadmap

This often leads to poor career decisions and dissatisfaction.

---

## 💡 Solution

Cognera AI solves this by:

* Using AI to analyze user preferences and behavior
* Recommending the most suitable career path
* Providing a clear roadmap for achieving that career
* Identifying skill gaps and suggesting improvements

---

## ⚙️ How It Works

1. User enters basic details or answers AI-generated questions
2. AI analyzes:

   * Interests
   * Skills
   * Personality traits
3. System matches user profile with career database
4. Provides:

   * Best career suggestion
   * Skill gap analysis
   * Personalized roadmap

---

## 🧠 AI Logic

* Dynamic question generation (AI asks relevant questions)
* Multi-dimensional analysis:

  * Analytical vs Creative
  * Government vs Private
  * Stability vs Salary
* Avoids generic answers (like always suggesting engineering)
* Produces **focused and realistic recommendations**

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/cognera-ai.git
cd cognera-ai
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Setup Environment Variables

Create a `.env` file:

```env
OPENROUTER_API_KEY=your_api_key_here
```

### 4. Run the Project

```bash
npm run dev
```
